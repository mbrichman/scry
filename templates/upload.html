How to Export ChatGPT Conversations

  1. Go to chat.openai.com
  2. Click on your profile picture or icon in the bottom-left corner
  3. Select "Settings"
  4. Click on "Data controls"
  5. Click "Export data"
  6. When you receive the export, unzip it and upload the conversations.json file here

Word Document Format

For Word documents, the tool will try to detect the structure automatically. For best results:

  • Start each user message with "You said:" on its own line
  • Start each ChatGPT message with "ChatGPT said:" on its own line
  • Include dates in standard format (YYYY-MM-DD) for better timeline filtering
